### 1. 변수란?
- 변수는 데이터를 담는 이름 붙은 저장 공간이다.
- `let`으로 선언하고, `=`로 값을 할당한다.

```javascript
let message;        // 선언
message = "Hello";  // 할당
let user = "Ann";   // 선언 + 할당을 한 번에
let a = 1, b = 2;   // 여러 개 한 번에 (권장하진 않음)
```

### 2. `var` (오래된 방식)
- `var`도 변수를 선언하지만 동작 방식이 달라 모던 코드에선 거의 쓰지 않는다.
- 이후 챕터에서 차이를 자세히 다룬다.

### 3. 변수 네이밍 규칙
- 문자, 숫자, `$`, `_`만 사용 (첫 글자는 숫자 불가).
- 대소문자를 구분한다 (`apple` ≠ `Apple`).
- 예약어(`let`, `return` 등)는 이름으로 못 쓴다.
- 관례: `camelCase`로 짓고, 의미가 드러나는 이름을 사용한다.

### 4. 상수 `const`
- `const`로 선언한 값은 재할당할 수 없다.

```javascript
const birthday = "18.04.1982";
// birthday = "..."; // ❌ Error
```

- 실행 전에 이미 알고 있는 고정값(하드코딩)은 대문자 + `_`로 쓰는 관례가 있다.

```javascript
const COLOR_RED = "#F00";
```
